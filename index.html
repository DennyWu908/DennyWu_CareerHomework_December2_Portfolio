Trivia Quiz

The Trivia game referred to in the above section. Users must complete this jQuery based quiz before time runs out. After the game finishes, it will return a player's number of correct, incorrect, and blank answers.

Gif Generator

A web application which uses the Giphy API to create animated images. It includes a number of default buttons to retrieve pictures of animals, but the program is not limited to these. Gif Generator can create buttons for almost any topic a user searches for.

Crystals Collector

In this game, players must use the crystal buttons to increase their score to match a randomly generated number. However, passing this number will result in a loss. Crystals Collector tests a player's judgment and memory.

LIRI (Language Interpretation and Recognition Interface)

A Node application inspired by Apple's SIRI. Users can input commands to check a Twitter account, obtain information on music from Spotify, or use OMDB to do the same for movies. LIRI retrieves this information using the APIs for each respective website. It can also read text files for input.
